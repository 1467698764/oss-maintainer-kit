# Examples

The `examples/minimal-python-project` directory is intentionally incomplete. It demonstrates how `omk` surfaces missing OSS maintenance basics.

Run:

```bash
omk report examples/minimal-python-project --format markdown
```

Then compare the findings with this repository, which includes README, license, tests, CI, contribution docs, templates, changelog, and security policy.

A generated sample is committed at [`docs/sample-report.md`](sample-report.md) so readers can preview report formatting without installing the package.
