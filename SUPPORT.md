# Support

OSS Maintainer Kit is maintained as a small open-source project. Please use the following channels so issues stay actionable.

## Questions

Open a GitHub Discussion or issue with:

- what you are trying to audit,
- the command you ran,
- the output you expected,
- the output you received.

## Bug reports

Use the bug report issue template. Include:

- operating system,
- Python version,
- OSS Maintainer Kit version or commit,
- minimal repository layout that reproduces the finding.

## Security reports

Do not open public issues for vulnerabilities. Follow `SECURITY.md`.

## Maintainer availability

This project is maintained by a solo maintainer. Clear reproductions, small pull requests, and documentation improvements are the fastest path to a response.
